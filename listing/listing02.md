Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

defer - это оператор который позволяет выполнить функцию в конце программы.
У нее есть 3 правила:
    - Аргументы отложенной функции вычисляются при вычислении оператора отсрочки
    - Если несколько функций вызываются с оператором defer, то те функции, которые вызываются раньше, будут выполняться позже всех
    - Отложенные функции могут читать и присваивать значения именнованным возвращаемым значениям

В первом вызове defer функция использует именованное возвращаемое знаение. Поэтому функция test вернет 2.
Во втором вызове defer функция не использует именованное возвращаемое значение. Она выполняется после команды return. Поэтому функция anotherTest вернет 1, а затем defer функция инкрементирует переменную x.
```
